### 技術的決定事項・ハマりポイント

- e.stopPropagation() on React synthetic event in ParamFormModal stops native event
  propagation at #root, preventing window.addEventListener('keydown') in SearchPalette
  from firing. This is the correct fix for the Esc-closes-palette bug.

- CodeMirror renders "- [ ] task" as a task widget (checkbox button). E2E テストの
  toContainText("- [ ] ...") は失敗するため、.cm-line セレクタで生ソース行を確認する
  アプローチに変更した。

- z-index 階層: .palette-backdrop(70) < .param-form-backdrop(75)。Playwright の
  pointer-events interception エラーを回避するために param-form-backdrop を
  dialog-backdrop クラスではなく専用クラスで実装。

- exact optionalPropertyTypes: description?: string | undefined と明示的に書く必要。
