## Precedence Rule

Token parsing order in resolveTemplate:
  1. Empty inner `{{}}` → verbatim
  2. Preset check: starts with `date:` or `now:` → resolvePreset
     (resolvePreset itself checks for `+Nd:FMT` / `-Nd:FMT` second-colon form)
  3. Pipe `|` present → `{{varName|fallback}}` fallback form
  4. Regular var lookup → verbatim + missing if absent
