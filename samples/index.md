# Loamium 機能サンプル集

Loamium の各機能を「実際に動く例」で説明するサンプル vault です。
`make samples` で vault (既定は `dev-vault/`) の `samples/` フォルダに投入されます。
**既存のファイルは上書きしません** — 自由に編集して実験してください。

> [!tip] まずはここから
> 一番のおすすめは [[dataview の使い方]] です。このサンプル集自体がクエリの実データになっています。

## 機能ガイド

- [[dataview の使い方]] — LIST / TABLE / TASK クエリの書き方と結果の読み方
- [[テーブル]] — Markdown テーブルと WYSIWYG 編集(セル編集・Tab/Enter 移動)
- [[embed と transclusion]] — `![[ノート]]` によるノート/セクション/画像の埋め込み
- [[callout と highlight]] — `> [!note]` ボックスと `==ハイライト==`
- [[数式 KaTeX]] — `$...$` / `$$...$$` の数式レンダリング
- [[mermaid 図]] — ```mermaid フェンスによる図の描画
- [[コードフェンス]] — シンタックスハイライト付きコードブロック
- [[タスク]] — `- [ ]` チェックボックスと dataview TASK
- [[wikilink とバックリンク]] — `[[リンク]]` とバックリンクパネル
- [[スラッシュメニュー]] — `/` から挿入できるコマンド一覧
- [[添付とアップロード]] — 画像/PDF のドラッグ&ドロップと `![[パス]]`
- [[テンプレート]] — `templates/` の雛形 + 変数(`{{...}}`)からノートを作成

## 同梱テンプレート(`templates/` に配置済み)

`make samples` で `templates/` に入るサンプルテンプレート。UI の「新規ノート ▸ テンプレートから」や `loamium new --template <名>` で使えます。

- **議事録** — 会議名・カテゴリ・参加者を入力 → `議事録/YYYY/MM/DD_会議名.md`
- **読書メモ** — 書名・著者・ステータス → `読書メモ/書名.md`
- **journal** — デイリージャーナル(`templates/journal.md`。毎日の自動生成に適用)

詳しくは [[テンプレート]] を参照。

## 使い方のヒント

| 操作 | 方法 |
| --- | --- |
| 検索 | `Ctrl+K`(ジャンプ)/ `/search` ページ(探索) |
| 保存 | 自動保存 + `Ctrl+S` |
| 挿入 | 行頭などで `/` を入力してスラッシュメニュー |
