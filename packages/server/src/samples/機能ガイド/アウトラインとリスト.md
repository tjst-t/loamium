# アウトラインとリスト

Loamium はリスト行(`- ` / `* ` / `+ ` の箇条書き、`1. ` / `1) ` の番号付き)を「アウトライナー」として扱えます。すべてピュアな Markdown なので、他のエディタでもそのまま開けます。

## インデント(アウトライン操作)

- リスト行で **Tab** を押すと、その項目(と子項目のサブツリー)を 1 段深くします
- **Shift+Tab** で 1 段浅くします
- 複数行を選択して押すと、選択に含まれる全リスト項目(とその子)を一括で上げ下げします
- 見出しや段落には効きません(リスト行だけ — C 方式)

番号付きリストの採番は、インデントを変えても自動で追従します(各階層で 1 から連番、子リストは 1 から再開)。

## リストタイプの一括変換(箇条書き ⇄ 番号付き)

箇条書きと番号付きは、いつでも相互に変換できます。**変換対象は「選択範囲が触れているリスト」**です(1 行にカーソルを置くだけでも、そのリストブロック全体が対象になります)。

### ショートカット(Google Docs / Word と同じ慣例)

| 変換 | ショートカット |
| --- | --- |
| 箇条書きに変換(`1. ` → `- `) | **Ctrl+Shift+8** |
| 番号付きに変換(`- ` → `1. `) | **Ctrl+Shift+7** |

### コマンドパレット(Ctrl+K)からも実行できます

`Ctrl+K` を開いて次のコマンドを選びます:

- **リストを箇条書きに変換**
- **リストを番号付きに変換**

### 変換の例(このリストで試せます)

1. まず番号付きリストにカーソルを置く
2. `Ctrl+Shift+8` で箇条書きへ
3. もう一度 `Ctrl+Shift+7` で番号付きへ戻す

- ネストした子項目も一緒に変換されます
    - この子項目も
    - 番号付きにすると 1. から採番されます
- インデント・チェックボックス・インラインフィールドは保持されます

### 変換のルール

- インデント(先頭の空白)・ネスト・子項目・マーカー後の空白はそのまま保持されます
- 番号付きへ変換したときは CommonMark のネスト規則で採番されます(各階層 1 から連番、子リストは 1 から再開)
- チェックボックスは温存されます: `- [ ] やること` を番号付きにすると `1. [ ] やること` になり、箇条書きに戻すと `- [ ] やること` に復帰します(相互変換で往復可能)
- 番号付きから箇条書きへ戻すとき、同じ階層にすでに `* ` や `+ ` の箇条書きがあればそのマーカーに合わせます(無ければ既定の `- `)

## エージェントからの変換(note_convert_list)

内蔵エージェント([[エージェントチャットの使い方]])に依頼して、ノートのリストタイプをまとめて変換することもできます(書き込み権限が必要です)。

- 「このノートのリストを全部番号付きにして」→ エージェントが `note_convert_list` ツールで一括変換します
- 行範囲(例: 3〜6 行目)を指定した変換もできます
- UI のショートカットと**同じ変換ロジック**を共有しているため、結果は一致します(ネスト採番・チェックボックス保持も同じ)
- 書き込みは監査ログ(`.loamium/audit.log`)に記録されます

## ポイント

- 正本は常にピュア Markdown。ブロック ID や独自記法は一切書き込みません
- 折りたたみ(子項目を持つリスト行のガター ▸)でサブツリーを畳めます
