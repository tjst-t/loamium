# コードフェンス

バッククォート 3 つのコードフェンスは、言語名に応じてシンタックスハイライトされます(Shiki)。

## TypeScript

```ts
interface Note {
  path: string;
  tags: string[];
}

export function isJournal(note: Note): boolean {
  return note.path.startsWith('journals/');
}
```

## Bash

```bash
# サンプルを vault に投入する(既存ファイルは上書きしない)
```
