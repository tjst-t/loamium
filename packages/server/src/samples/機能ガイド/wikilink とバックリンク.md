# wikilink とバックリンク

`[[ノート名]]` でノート同士をリンクできます。`[[` と入力するとオートコンプリートが開きます。

## 基本

- 通常のリンク: [[index]]
- 表示名を変える(エイリアス): [[index|サンプル索引へ]]
- 見出しへのリンク: [[dataview の使い方#TASK — チェックボックスを横断で集める]]

## 壊れリンク

存在しないノートへのリンクは赤い破線で表示されます: [[まだ存在しないノート]]
クリックするとその名前で新規ノートを作成します。

## バックリンク

右サイドバーの「バックリンク」タブに、**このノートへリンクしている参照元**が文脈付きで表示されます。
このサンプル集では [[index]] など多くのノートが相互にリンクしているので、いろいろなノートで確認してみてください。

## リネーム追従

ノートをリネームすると、vault 全体の `[[旧名]]`(`[[旧名#見出し]]` / `[[旧名|別名]]` を含む)が自動で書き換わります。コードフェンス内は変更されません。
