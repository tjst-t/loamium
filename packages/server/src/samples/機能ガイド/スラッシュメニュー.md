# スラッシュメニュー

エディタで `/` を入力すると、挿入コマンドのメニューが開きます(文字で絞り込み、↑↓ で選択、Enter で挿入、Esc で閉じる)。
記法を暗記しなくても、ここから発見して挿入できます。挿入結果はすべて標準 Markdown です。

## コマンド一覧

| コマンド | 挿入されるもの | 関連ガイド |
| --- | --- | --- |
| `/table` | テーブル(サイズをグリッドで選択 → 最大 8×8) | [[テーブル]] |
| `/callout` | `> [!note]` ボックス | [[callout と highlight]] |
| `/code` | コードフェンス | [[コードフェンス]] |
| `/mermaid` | mermaid フェンス | [[mermaid 図]] |
| `/dataview` | dataview クエリフェンス | [[dataview の使い方]] |
| `/checkbox` | `- [ ]` タスク | [[タスク]] |
| `/heading` | 見出し | |
| `/date` | 今日の日付 | |

## ポイント

- コードフェンスの中やインラインコード内では開きません(誤爆防止)
- `/table` は「3×3」などのサイズをグリッドで選んでから挿入します(↑↓←→ でサイズ変更)
