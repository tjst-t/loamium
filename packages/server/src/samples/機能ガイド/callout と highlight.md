# callout と highlight

## callout(注記ボックス)

引用行の先頭に `[!タイプ]` を付けると、色付きの注記ボックスになります(Obsidian 互換)。

> [!note] ノート
> 基本の注記。タイプ名は大文字小文字どちらでも構いません。

> [!info] 情報
> 補足情報に。

> [!tip] ヒント
> ちょっとしたコツに。

> [!warning] 注意
> 気をつけてほしいことに。

> [!danger] 危険
> 破壊的な操作の警告に。

`[!note]-` のようにマイナスを付けると**折りたたみ式**になります(タイトルをクリックで開閉)。

> [!note]- 折りたたみ callout
> ここは畳まれた中身です。クリックで開きました。

## highlight(ハイライト)

`==` で囲んだテキストは ==このようにハイライト== されます。
カーソルを同じ行に置くとソース記法が見えます(Live Preview)。

> [!tip]
> どちらも標準的な Markdown テキストのまま保存されるので、他のエディタでも壊れません。
