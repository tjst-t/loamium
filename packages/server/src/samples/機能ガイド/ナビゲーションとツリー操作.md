# ナビゲーションとツリー操作

Loamium の画面移動は、すべて **URL(履歴)** に紐づいています。ブラウザの戻る/進む、ヘッダの `<` `>` ボタン、ブックマークがどの画面でも効きます。左サイドバーのツリーは、ワンタッチで全展開・全折りたたみできます。

## ヘッダの戻る/進む(< >)と URL

各画面の左上ヘッダに `<`(戻る)/ `>`(進む)ボタンがあります。開いた順に履歴へ積まれるので、行き来が自由にできます。

| 画面 | URL |
| --- | --- |
| ノート | `/n/フォルダ/ノート名`(`.md` は省略) |
| すべてのファイル | `/files` |
| 詳細検索 | `/search?q=...&tag=...` |
| 設定(メニュー) | `/settings` |
| 設定(各項目) | `/settings/general` / `/settings/agent` / `/settings/templates` など |

- URL を直接開く/リロードすると、その画面に直接着地します。
- ヘッダのパンくずが現在のルートを表示します。

## 設定メニューのナビゲーション

設定にも `<` `>` 付きのヘッダが付き、各項目が URL(`/settings/<項目>`)になります。項目を切り替えるたびに履歴へ積まれるので、**戻る/進むで設定項目間を移動**できます。

- **全体 / エージェント / プライバシー** — 左のサブメニューを表示したまま切り替えます。
- **コンテンツ群(テンプレート・スマートフォルダ・スマートコマンド・エージェントジョブ・タスク語彙)** — 開くと左のサブメニューが隠れ、詳細が全幅で開きます。**戻る(`<`)でメニューへ戻れます**。

### 右サイドバーの自動折りたたみ

設定に入ると、右サイドバー(インフォ / Agent)が開いていれば **自動で折りたたまれ**、編集領域を広く使えます。設定から抜けると **元の状態が復元**されます(自分で閉じていたら閉じたまま)。

## ツリーの全展開 / 全折りたたみ

左サイドバー上部の「ノート / スマート」タブの右側に、2 つのボタンがあります(ノート・スマート両ビュー共通)。

| ボタン | 動作 |
| --- | --- |
| すべて展開 | ツリーのフォルダ(スマートフォルダ)をすべて開きます |
| すべて折りたたむ | ツリーをすべて閉じ、トップ階層だけを表示します |

- 深いフォルダ構成でも、目的の階層をすばやく見渡せます。
- 個別のフォルダはこれまで通り、フォルダ行のクリック(シェブロン)で開閉できます。

> [!note] モバイル表示
> 画面幅が狭いときは、設定はドリルダウン(メニュー → 項目 → 戻る)で操作します。左サイドバーはオーバーレイとして開きます。
