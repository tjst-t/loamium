# 添付とアップロード

画像や PDF などのファイルは、エディタへ**ドラッグ&ドロップ**するか、クリップボードから**ペースト**するだけでアップロードできます。

## 動き

1. ファイルをエディタにドロップ(またはスクリーンショットをペースト)
2. `assets/` に保存され、カーソル位置に `![[assets/ファイル名]]` が挿入される
3. 同名ファイルがある場合は `名前-1.png` のように自動リネームされる(トーストで通知)

## 埋め込みプレビュー

- 画像はその場に表示されます
- PDF は `![[file.pdf]]` でページプレビュー、テキストファイルは冒頭プレビューになります
- プレビューできない形式はダウンロードカードになります

## ファイルの管理

サイドバー下の「すべてのファイルを表示」(`/files` ページ)で、添付の一覧・プレビュー・リネーム・削除ができます。
リネームすると本文中の `![[リンク]]` も自動で追従します。詳しくは [[wikilink とバックリンク]] を参照。
