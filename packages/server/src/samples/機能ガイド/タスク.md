# タスク

`- [ ]` / `- [x]` のチェックボックス付きリストが使えます。表示上のチェックボックスをクリックすると、ファイルの `[ ]` / `[x]` が書き換わります。

## 今日のタスク(サンプル)

- [x] サンプル集を vault に投入する(`make samples`)
- [ ] [[dataview の使い方]] を読む
- [ ] 自分のノートで TASK クエリを書いてみる
  - [ ] ネストしたタスクも書けます

## ノート横断でタスクを集める

タスクは dataview の TASK クエリでノート横断で一覧できます。下の例はこのサンプル集のプロジェクトノートのタスクを集めます(チェックすると元ファイルが更新されます)。

```dataview
TASK FROM #sample-project WHERE !completed
```

## ポイント

- リスト行では Tab / Shift+Tab でインデント(アウトライン操作)が効きます
- 見出しや段落には効きません(リスト行だけ — C 方式)
