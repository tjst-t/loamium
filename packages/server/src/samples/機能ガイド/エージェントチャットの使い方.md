# エージェントチャットの使い方

Loamium に内蔵されたエージェントは、vault 内の情報を自然言語で調べたり、ノートの作成・編集を依頼できる対話型 AI です。

> [!info] 設定が必要
> エージェントを使うには `.loamium/agent.json` に API キーを設定する必要があります。詳細は `agent.json` の設定例を参照してください。

## 基本的な使い方

右側の「Agent」タブをクリックするとチャット画面が開きます。下部の入力欄にメッセージを入力して Enter で送信できます。

- **Enter** — 送信
- **Shift+Enter** — 改行
- **中断ボタン (■)** — 応答生成を途中で止める（部分応答は画面に残る）

## メッセージを編集して会話をやり直す

過去のメッセージにマウスを重ねると、**鉛筆アイコンの編集ボタン**が表示されます。

1. 編集したいユーザーメッセージにホバーして鉛筆ボタンをクリック
2. 入力欄に元のテキストが入るので、内容を修正する
3. Enter で再送信 — **そのメッセージより後の会話履歴が破棄**され、修正後の内容から再生成される

キャンセルしたい場合は **Esc キー** または「キャンセル」ボタンで元に戻せます。

> [!tip] 軌道修正に便利
> 質問の仕方を変えたい場合や、方向性を変えたい場合に新規セッションを作り直す必要なく、その場でやり直せます。

## セッションの管理

会話はセッション単位で保存されます。

- セッションバーの **▼ ボタン** — 過去のセッション一覧を表示・切り替え
- **+ ボタン** — 新規セッションを開始
- **🛡 権限ボタン** — エージェントが使える操作範囲（読み取り専用 / ノート書込 / フル）を変更

## 現在開いているノートをコンテキストとして使う

エディタでノートを開いた状態でメッセージを送ると、そのノートのパスが自動的にコンテキストとして渡ります。

- 「この文書を要約して」— 開いているノートを読んで要約
- 「今のノートに今日の日付の見出しを追加して」— 現在ノートを直接編集 (書き込み権限が必要)
- ノートを開いていない場合はコンテキストなしで通常通り動作

## エージェントが書いたらエディタに自動反映

エージェントがノートを書き込んだとき、そのノートがエディタで開かれていれば **明示的なリロードなしに内容が最新化**されます。

- 未編集の場合: 自動でエディタ内容を差し替え
- 編集中の場合: 未保存の編集を保護し、更新があった旨をトーストで通知

## エージェントができること（権限による）

| 権限 | 操作 |
|------|------|
| 読み取り | ノート検索・読み取り・タグ一覧・バックリンク調査 |
| ノート書込 | ノート作成・編集・ジャーナル追記・テンプレート適用 |
| フル | 上記すべて + コマンド実行・スマートフォルダ編集 |

操作はすべて `.loamium/audit.log` に記録されます。

## 新規セッションの既定権限を変える

毎回手動で権限を変えるのが面倒な場合は、**設定 → 全体 → 既定 → 「Agent 新規セッションの既定権限」** から初期権限を変更できます。

| 選択肢 | 説明 |
|--------|------|
| read-only（既定） | 読み取りのみ。誤操作を防ぎたい場合に |
| notes-rw | ノートの読み書き・ジャーナル追記。日常的な編集作業に |
| full | 全ケーパビリティ。コマンド実行・スマートフォルダ管理も含む |

> [!note] セキュリティ保証
> 既定を `full` に変えても、機密領域 deny リストや自己昇格防止は常に有効です。
> サーバーの LOAMIUM_MODE によるクランプも変わらず適用されます。

設定は `system/settings.yaml` の `agentDefaultPreset:` キーに保存されます（git 管理可能）。
