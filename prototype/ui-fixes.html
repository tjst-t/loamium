6 Ctrl-B でエディタの Bold/UnBold をトグル S2e8a4c-6

CodeMirror コマンド toggleBold を Mod-b (Ctrl-B / Cmd-B) にバインド。ピュア Markdown (**text**) のみ使用。

1 テキストを選択:   重要な設計方針
2 Ctrl + B → Bold 化:   **重要な設計方針**
3 Ctrl + B (再度) → UnBold:   重要な設計方針   (** を除去)
4 選択なし (カーソルのみ):   Ctrl + B → **** を挿入してカーソルを内側に配置

実装: packages/ui/src/bold-toggle.ts に toggleBold: Command。 選択が **...** で囲まれているか正規表現で判定。 Editor.tsx の keymap.of に {'{ key: "Mod-b", run: toggleBold }'} を追加。
