Sa10026-4 注記
既定は非表示。トグル ON で system/ を表示。定義ファイル(.yaml / テンプレは .md)はクリックで編集エディタで開く。ドラッグ&ドロップで並べ替えると各ファイルの order: を安定ソートで再採番(PUT /api/notes/…)。system/ はディスク上は可視で .gitignore 対象外(.loamium とは別)。

スマートフォルダ / コマンドは YAML、テンプレートは Markdown

例: system/smart-folders/journal.yaml は純 YAML(title / order / icon / query)。DQL は query: フィールドに文字列で持ちます。テンプレートはノート本文そのものなので system/templates/journal.md(frontmatter + 本文)のまま。

この画面は Sa10026-4(ツリー表示トグル)のプロトタイプです。統一設定画面は settings.html。
