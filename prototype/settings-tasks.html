タスク語彙設定

保存済み

タスクの ステータス と 優先度 の語彙を定義します — system/settings.yaml の tasks: ブロックに保存され、 スラッシュメニューの タスク / TODO ポップオーバーや Dataview の列ヘッダー・ピル表示で使われます。

プロトタイプ注記 (Se3b7a2-8 / ADR-0026)
保存先: system/settings.yaml の tasks: ブロック(vault ファイル・agent 編集可)。 保存経路は型付き設定 API (zod + REST PATCH /api/settings/tasks)経由。 変更はスラッシュメニューポップオーバーと Dataview のピル表示にリアルタイム反映。 「完了扱い」ON のステータスのみ [x] チェックボックスとして書き込まれ、残りは [ ]。
YAML プレビュー(読み取り専用) system/settings.yaml 
# system/settings.yaml — tasks section
tasks:

  statuses:
    - key:   todo
      label: TODO
      color: '#a3a4ad'
    - key:   progress
      label: 進行中
      color: '#6e56cf'
    - key:   blocked
      label: ブロック
      color: '#d64545'
    - key:   done
      label: 完了
      color: '#2f9e63'
      done:  true   # → [x] checkbox
    - key:   cancelled
      label: キャンセル
      color: '#75767f'

  priorities:
    - key:   highest
      label: 最高
      color: '#d64545'
    - key:   high
      label: 高
      color: '#e07c2e'
    - key:   medium
      label: 中
      color: '#b7791f'
    - key:   low
      label: 低
      color: '#2f9e63'
このプレビューは入力に連動してリアルタイム更新されます。 保存ボタンで PATCH /api/settings/tasks を呼び、このファイルに書き込まれます。
