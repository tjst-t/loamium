案A: Obsidian 方式。 chevron がテキストの直前 (4px 間隔) に来るため「テキスト行の付属物」として自然に読める。 ガター列が不要になりエディタ幅を最大化できるが、CodeMirror GutterMarker API を使わず Decoration (WidgetDecoration) で行頭に埋め込む実装に切り替える必要がある。

案B 詰めたガター — 薄い左ガターをコンテンツに密着させる

案B: 詰めたガター方式。 GutterMarker API は現行のまま維持。ガター幅を 36px → 22px に縮小し、コンテンツの左 padding を 4px に揃えることで chevron とテキストの間隔を約 26px 削減する。 実装変更は CSS 調整のみで最小だが、ガターは常に一定列に固定されるため、 H1 の大きなフォントに対して chevron がやや小さく見えるという視覚的な不均衡は残る。
