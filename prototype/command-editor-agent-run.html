② agent-run を選択 → 雛形が挿入される

prompt 必須 + 任意フィールドをコメント提示

PROTO NOTE KIND_SCAFFOLDS['agent-run'](AC-S5a66e4-5-2)。雛形は prompt(必須)を含み、 permissions? / maxTurns? / timeoutSec? / open? / when? / when-not? を 任意フィールドのコメント行として提示します。挿入直後の YAML はそのまま有効(prompt 以外は任意)で、カーソルは prompt: の値に置かれます。

scaffold 挿入後 — 有効な YAML(prompt 入力待ち)
commands / summarize-note
⚡ スマートコマンド定義 ✓ 有効
YAML frontmatter · agent-run 雛形を挿入
loamium-command: name: ノートを要約 description: 選択ノートをエージェントで要約し「まとめ」に追記 params: - name: notePath label: 対象ノート required: true steps: - kind: agent-run prompt: "{{notePath}} を 5 行以内で要約して" # open: true # 実行後に結果ノートを開く (任意) # permissions: read-only # read-only | append-only | full (任意, 既定 read-only) # maxTurns: 8 # エージェントの最大ターン数 (任意) # timeoutSec: 120 # タイムアウト秒 (任意) # when: "{{notePath}}" # 値があるときだけ実行 (任意) # when-not: "" # 値が空のときだけ実行 (任意)
