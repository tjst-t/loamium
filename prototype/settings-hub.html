スマートフォルダ

DQL クエリ / ピンで作る動的フォルダ — system/smart-folders/*.yaml。ドラッグで並べ替え。

📓
Daily Journal
LIST FROM "journals" SORT file.name DESC LIMIT 14
journal(N)tag(#…)folder("…")custom
✅
Todos
TASK WHERE !completed
🕒
Recent
LIST SORT file.mtime DESC LIMIT 10
⭐
Bookmarks
LIST WHERE bookmark = true
