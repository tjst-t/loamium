プロトタイプ注記(S8a3f2e-4 / ADR-0025・0026)
external バックエンド選択時の状態です。既存の baseUrl / API キー($ENV)/ 接続テスト / モデル一覧はそのまま。 backend: "external" と選択モデルは PUT /api/settings/agent/connection(shared zod 拡張)で .loamium/agent.json に保存されます。 バックエンドは自動フォールバックせず、ユーザーが明示選択します。選択中のバックエンドが未準備(external だがキー無し / local だがモデル未選択)ならエージェント接続は無効です。 ?mode=ro を URL に付けると read-only 無効化状態を確認できます。
ローカルモデル選択時は settings-llm-backend-local.html / 未取得時は settings-llm-backend-empty.html を参照。
